7. Grepping in MML

C-c g

mizar-grep-abs

C-c C-g

mizar-grep-full

Command: mizar-grep-abs exp

Grep MML abstracts for regexp exp.
Variable ‘mizar-grep-case-sensitive’ controls case sensitivity. The results are shown and clickable in the Compilation buffer.

Command: mizar-grep-full exp

Greps full MML articles for regexp exp.
Variable ‘mizar-grep-case-sensitive’ controls case sensitivity. The results are shown and clickable in the Compilation buffer.

User Option: mizar-grep-case-sensitive

Tells if MML grepping is case sensitive or not.

The default value is t.
